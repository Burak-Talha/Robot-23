# Robot-23
7682 Charged Up season code

This repository contains odometry based setpoint calculations and improved user friendly mode scenarios.

## Features
### Auto
**Autonomous Section contains autonomous path planning, mode selector and utils class**
### Control Panel
**Designed for improving user experience in controlling tele operated mode**
### lib
**Contains 7682, 254 upgraded libs for control scenarios, odometry and more than this**
### Subsystems
**Subsystems have systems in robot(like gripper).**
#### Vision
*Vision subsystem based on PhotonVision library. And it'll give best pose estimation on the field based on april tag.*
#### Super Structure
*Super Structure controlling pid setpoint, organizing subsystems based on choosed mode for robot.*
#### Infrastructure
*This section returns info about pneumatic systems, power distribution panels*
